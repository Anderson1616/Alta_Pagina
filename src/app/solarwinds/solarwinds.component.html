SOLARWINDS

Nuestro negocio está diseñado para ayudar a los profesionales de tecnología (profesionales de operaciones de TI, profesionales de DevOps y proveedores de servicios administrados) a fin de resolver los retos de TI de la actualidad.

Nuestra relación y vinculación con ellos nos brinda información y comprensión de las que dependemos para ofrecer productos diseñados para resolver los problemas de administración de TI del modo que los profesionales de tecnología quieren que se resuelvan. Fomentamos esta relación mediante nuestra interacción diaria en THWACK, nuestra comunidad en línea con más de 150 000 miembros registrados.

MONITOREO DE TI INTEGRANDO LA PLATAFORMA ORION

La plataforma Orion es una arquitectura de software modular diseñada para ofrecer a los departamentos de TI visibilidad unificada, escalabilidad empresarial y uso sencillo de la consola de gran capacidad de ampliación.

NETWORK PERFORMANCE MONITOR

Monitoreo de errores, disponibilidad y desempeño para redes de todos los tamaños
  • Acelera la resolución de problemas, soluciona los cortes en la red y contribuye a reducir el tiempo de inactividad.
  • Alertas inteligentes personalizables con reconocimiento de dependencias y topología de red.
  • Mapas térmicos inalámbricos, pronóstico automatizado de capacidad alertas e informes.

NETFLOW TRAFFIC ANALYZER

Utilización del ancho banda y el tráfico de red en un panel fácil de ver
  • Monitoree los patrones de tráfico y el ancho de banda de la red a nivel de interfaz con una granularidad de hasta un minuto.
  • Uso del ancho de banda por usuario, aplicación, protocolo y grupo de direcciones IP.
  • Analice los patrones del tráfico a lo largo de meses, días o minutos al explorar cualquier elemento de la red en profundidad.

NETWORK CONFIGURATION MANAGER

Software de administración centralizada de configuraciones y cambios de dispositivos de red.
  • Copias de seguridad, comparaciones y reversiones automáticas.
  • Auditorías y detección de cambios de configuración en tiempo real para el control de cumplimiento.
  • Cambios de configuración de implementación en masa.
  • Análisis e identificación de vulnerabilidades de dispositivos de Cisco.

IP ADDRESS MANAGER

Software de administración de direcciones IP de uso sencillo.
  • Rastreo de direcciones IP, alertas e informes automatizados.
  • Administración integrada de direcciones IP, DHCP y DNS.
  • Rastreo de historiales y detalles de IO.

NETWORK OPERATION MANAGER

Software de administración de redes unificado para entornos de TI locales, en la nube e híbridos.
  • Monitoreo, detección y solución de problemas de desempeño de redes.
  • Monitoreo de tráfico, ancho de banda y WAN.
  • Rastreo de dispositivos y usuarios.

NETWORK AUTOMATION MANAGER

Software de administración de operaciones y automatización de redes unificado.
  • Monitoreo, detección y solución de problemas de redes avanzados.
  • Monitoreo de tráfico, ancho de banda y WAN.
  • Administración de cambios y configuraciones.
  • Alta disponibilidad.

SERVER &APPLICATION MONITOR

Administre, detecte y solucione problemas de desempeño y disponibilidad de las aplicaciones
  • Gran visibilidad de aplicaciones, bases de datos, servidores de host, virtualización y sistemas de almacenamiento de forma local y también en la nube.
  • Compatibilidad inmediata con más de 1200 aplicaciones, que incluye Exchange, SQL Server, etc.
  • Planificación de capacidad e inventario de archivos de software.

DATABASE PERFORMANCE ANALYZER

Obtenga ayuda para resolver problemas complejos en la base de datos y optimice los tiempos de respuesta de las aplicaciones.
  • Compruebe exactamente lo que está afectando el desempeño de su base de datos y obtenga asesoramiento de expertos para resolverlo.
  • Correlacione el tiempo de respuesta con los recursos de VM, el huésped y la entrada/salida del almacenamiento con el historial y las líneas de base.
  • Arquitectura sin agente ideal para prueba, desarrollo y producción de forma local en la base.

VIRTUALIZATION MANAGER

Rendimiento, detección y solución de problemas, capacidad y administración de la expación desde la VM hasta el almacén de datos.
  • Visibilidad del uso y el desempeño de las máquinas virtuales (VM), los hosts, los clústeres y los almacenes de datos.
  • La inteligencia procesable superior permite correcciones de varios pasos con un solo clic para la asignación activa y prevista de recursos y desempeño de VM.
  • Desconecte las VM activas, ajuste recursos sobre y suba signados y resuelva problemas de Co-stop generalmente minutos.

STORAGE RESOURSE MONITOR

Monitoreo de desempeño y capacidad de almacenamiento multiproveedor
  • Diseñado para brindar visibilidad de desempeño en tiempo real de LUN de arreglos SAN y NAS, grupos de RAID, volúmenes, discos, etc.
  • Planificación automatizada de la capacidad de almacenamiento para ayudar a identificar zonas altas, horas pico y posibles interrupciones.
  • Identifique problemas de desempeño en la contención de LUN y otras dificultades de almacenamiento.

WEB PERFORMANCE MONITOR

Evalúe la experiencia del usuario, detecte y solucione la latencia para las aplicaciones web
  • Visibilidad de la experiencia del usuario final para transacciones de aplicaciones web y SaaS.
  • Monitoree el desempeño de las aplicaciones de diversas ubicaciones geográficas más allá del servidor y seguridad.
  • La integración incorporada a la plataforma de Orion puede mostrar las dependencias de infraestructuras para ayudar a lograr una detección y solución de problemas más rápida.

SERVER CONFIGURATION MANAGER

Supervisión de cambios en aplicaciones y sistemas de Windows fácil de usar
  • Evalúe los cambios en sistemas y aplicaciones Windows con respecto a líneas base personalizadas.
  • Correlacione los incidentes de rendimiento con respecto a los cambios de configuración.
  • Detecte y rastree cambios en el transcurso del tiempo.
